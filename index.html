Motivation

With having a solar grid 5.2 kW, a house battery, a Mitsubishi Eclipse PHEV and an e-Charger in the past I was manually controlling the car charging - which was quite annoying after a while.

With some investigations I have installed Solar Charging evcc - https://evcc.io - which is working perfect and up to now 97% of the car charging is provided by the sun : - )  The evcc UI is great and works perfect on my iPad and iPhone.

In certain situations I would like to have the solar production, consumption and charging data with a finger tip on my iWatch.
Unfortunately or lucky me there is no iWatch app providing all the functionality that I want. Then I made it... please see below.

SolVecConMon

SolVecConMon stands for Solar Power and Vehicle Control/Monitoring. Have all important solar and loadpoint values on your Apple Watch with a finger tip - IN REAL TIME. It's a pure Apple Watch app! There is no Apple iPhone companion app - configuration also done in SolVecConMon.


Please note - prerequisite is to have installed Solar Charging evcc - https://evcc.io - SolVecConMon is powered by evcc!

 

Track IN REAL TIME

  • solor power generation
  • house consumption
  • house batterie charging/discharging
  • Vehicle(loadpoint) monitoring
  • Vehicle(loadpoint) charge mode control

 

This version supports

  • one solor grid
  • one house battery
  • one loadpoint (a second loadpoint wouldn't be shown)

 

The application is working in your local network with running a local installation of https://evcc.io. For that you need to enter your evcc installation server in the App's Configuration tab like http://mymachine:7070 or http://myIP:7070 (expected you are using the default evcc port 7070).

 

If you want the App having access outside your local network you need additionally setup an 'Edge Router' (reverse proxy), which provides basic authentification feature and https secure connection. I'm running it on a RaspberryPi 4 (LAN connected), with below software

  • Image/Container Management Docker - https://docs.docker.com - open source
  • Solar Charging evcc container - https://evcc.io - open source
  • Edge Router traefik proxy - https://doc.traefik.io - open source
  • Let's Encrypt - https://letsencrypt.org - a nonprofit Certificate Authority

Please find configuration details there: SolVecConMon Server Setup Description

With that SolVecConMon is additionally powered by Raspberry, Docker, Traefik and Let's Encrypt.

Privacy statement

In order to download your solar and charging data from your evcc installation and to control the charge mode this app is communicating with your evcc server.

The app isn't storing any of that communication!

In case you have setup a server so that the App has access from outside your local network the Basic Authentication credentials are temporarily stored in the app.
